#+TITLE: Eigenvalue Configurations
Algorithms to compute quantifier-free conditions on the entries of real symmetric matrices F and G so that their eigenvalues are arranged in a given way. See [1], [2].

[[https://www.maplesoft.com/products/Maple/][Maple]] code is available now. Python implementation coming soon!

** Usage
*** Maple
#+begin_src maple
  with("EC.mpl"):
  with(LinearAlgebra):
  F := DiagonalMatrix([0,1]):
  G := DiagonalMatrix([0,2]):
  Csig, Asig := ConditionsSig(F,G);
  Csym, Asym := ConditionsSym(F,G);
#+end_src

** References
1. Hong, Hoon, Daniel Profili, and J. Rafael Sendra. "Conditions for eigenvalue configurations of two real symmetric matrices (symmetric polynomial approach)." [[https://arxiv.org/abs/2401.00089][arXiv preprint arXiv:2401.00089]] (2026).

2. Hong, Hoon, Daniel Profili, and J. Rafael Sendra. "Conditions for eigenvalue configurations of two real symmetric matrices (signature approach)." [[https://arxiv.org/abs/2401.00866][arXiv preprint arXiv:2401.00089]] (2026).
