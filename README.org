#+TITLE: Eigenvalue Configurations

** Usage

*** Maple
#+begin_src maple
  with(EC.mpl):
  F := DiagonalMatrix([0,1]):
  G := DiagonalMatrix([0,2]):
  Csig, Asig := ConditionsSig(F,G);
  Csym, Asym := ConditionsSym(F,G);
#+end_src
